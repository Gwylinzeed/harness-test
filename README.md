# RetroForge MVP (Web)

这是基于需求文档 `docs/requirements/retroforge-v1.0.0.md` 实现的首批 MVP 原型。

## 已实现范围
- FR-1 项目创建与模板启动（空白/平台跳跃/地牢模板）
- FR-2 基础场景编辑（20x12 网格绘制）
- FR-2 图层管理与碰撞编辑（基础层/装饰层/碰撞层）
- FR-3 角色/对象属性编辑（速度、生命值、动画状态、触发器）
- FR-4 可视化逻辑编排（ECA 规则：按键触发 -> 动作执行）
- FR-5 资源管理（本地文件导入并维护资源清单）
- FR-6 一键预览与事件日志
- FR-7 版本快照与回退（本地快照）
- FR-8 项目导出（JSON 文件 / 单文件 HTML 包）
- FR-8 项目导入（支持导入既有 JSON 项目并继续编辑）
- FR-9 新手引导（分步完成与进度状态）
- FR-10 协作反馈（只读分享链接 + 评论）
- NFR-5 基础自动保存与恢复（localStorage）

## 本地运行
1. 直接用浏览器打开 `src/index.html`
2. 输入项目名并选择模板，点击“创建项目”
3. 在编辑器区域选择基础层、装饰层或碰撞层后进行绘制，左键绘制、右键擦除
4. 在“资源管理”中选择文件并导入，支持按名称或类型搜索资源
5. 在“版本快照”中创建快照，并可从下拉列表选择任意快照回退
6. 在“新手引导”中开始并推进步骤，观察引导进度
7. 在“协作反馈”中生成只读分享链接并添加评论，可通过 `src/share.html?share=<token>` 只读查看
8. 可先点击“导入项目”选择之前导出的 JSON，直接恢复到可编辑状态
9. 选择导出格式后导出 JSON 或可直接打开的 HTML 包
10. 点击“运行预览”后可用方向键或 WASD 控制角色，并触发金币/敌人/逻辑规则；支持暂停/恢复/重置与实时调试信息
11. 点击“加载最近项目”恢复自动保存数据

## 说明
- 当前为前端静态原型，不依赖后端。
- 项目导入/导出采用 `schemaVersion` 字段，当前导出版本为 `1.1.0`，兼容导入 `1.0.0` 与 `1.1.0`。
- 下一批次可继续实现：更多模板类型、多人协作与云端同步。
